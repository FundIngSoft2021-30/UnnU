## Instalar y ejecutar la app
### usamos react js  y node js 16.2.0 
 para instalar todos los componentes que vas a necesitar para este proyecto 
```nodejs
npm install 
npm install react-router-dom
npm install react-router-dom
```
si quieres ver la app en
```nodejs
npm start
```
